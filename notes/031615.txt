03/16/15
========

Reanalysis of samples generated by CMain and KCoyne will be assembled to generate a single reference transcriptome for future analysis.
Samples were generated using H. akashiwo CCMP2393 (https://ncma.bigelow.org/ccmp2393#.VQbQKxDF-ZQ)


Quality Trimming
================

Samples were quality trimmed using trimmomatic-0.33 (see qc2.py)
The included adapter sequences from trimmomatic were used (TruSeq3-SE.fa)
Samples were combined and run separately for quality trimming.


Trinity De Novo Assembly
========================

Combined fastq, H_akashiwo_qc.fastq, was run using the Trinity software package.  Due to the large number of sequences.  Samples will be normalized (--normalize_reads)


Command:
	chris@Lachesis:/media/transcriptome$ trinity --seqType fq --max_memory 30G --CPU 4  --normalize_reads --single qc/H_akashiwo_qc.fastq
	Trinity version: v2.0.6
